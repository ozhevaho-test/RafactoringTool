# Refactoring Tool

Інструмент для рефакторингу C++ коду

## Функціональність

### Реалізовано:
- [x] Rename Variable - перейменування змінних

### Заплановано:
- [ ] Extract Method - винесення коду в метод
- [ ] Introduce Constant - винесення константи

## Команда розробників
- Розробник 1 - Team Lead, Rename Variable
- Розробник 2 - Extract Method
- Розробник 3 - Introduce Constant

## Як використовувати
1. Clone репозиторій
2. Відкрийте в Visual Studio
3. Запустіть програму (F5)
